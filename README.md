# TrackWasher for Pinokio (v3.3)

One-click installer for [TrackWasher](https://github.com/mikecastrodemaria/TrackAICleaner) — pre-mastering & audio enhancement for AI-generated music.

## Install

1. Install [Pinokio](https://pinokio.computer/) if you don't have it
2. In Pinokio, go to **Download**
3. Paste this URL:

```
https://github.com/mikecastrodemaria/trackwasher.pinokio.git
```

4. Click **Install**, then **Start**

That's it. No terminal, no Python setup needed.

## What is TrackWasher?

A 12-stage audio processing tool that enhances and pre-masters AI-generated music for release:

**Audio Enhancement** — stereo depth, stereo width, HF refinement, harmonic enrichment, timing humanizer, ambience shaping

**Pre-Mastering** — multiband compressor, tape saturation, glue compressor, Mid/Side EQ, soft clipper, LUFS normalization

Includes platform presets for Suno, Udio, and more. Before/after spectrogram comparison. Supports WAV, FLAC, MP3, OGG input.

## Menu

| Button | Action |
|---|---|
| **Install** | Downloads TrackWasher + installs Python dependencies |
| **Start** | Launches the Streamlit web UI |
| **Update** | Pulls the latest version from GitHub |
| **Reset** | Removes everything for a clean reinstall |
